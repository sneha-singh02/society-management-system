Society Management System
-A role-based Society Management System built using Django that allows Admin and Owners to manage and view society-related information efficiently.

Features
-Role-based authentication (Admin & Owner)
-Admin Dashboard
-Owner Dashboard
-Manage Owners
-Form handling using Django Forms
-Template-based UI
-Login & Logout functionality

Tech Stack
-Python
-Django
-SQLite (default database)
-HTML, CSS
-Django Templates

Setup Instructions

Follow these steps to run the project locally:

1.Clone the repository
2.Create Virtual Environement
3.Install Dependencies
4.Apply Migrations
5.CreateSuperuSer(if neede)
6.Run Server
